# Robot Builder
Unity project for modular robot construction.

## Short description
Robot Builder is a Unity project focused on modular part assembly and weight-based physics.

## Key features
* **Modular Assembly:** ScriptableObject-based system for swapping heads, torsos, and legs.
* **Physics Simulation:** Dynamic jump force calculations based on the robot's total mass and power.
* **Orbit Camera:** Interactive camera system for 360° model inspection.
* **Architecture:** Clean separation between Core logic and UI management.
